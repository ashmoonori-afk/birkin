# Birkin Agent

**AI agent platform for orchestration, cybersecurity management, and lightweight web UI environments.**

> Forked from [Hermes Agent](https://github.com/NousResearch/Hermes-Agent) by Nous Research (MIT License).

Birkin extends the Hermes Agent foundation with a focus on:
- AI agent orchestration and harness management
- Cybersecurity management workflows
- Lightweight AI-powered web UI environments
- Accessibility for non-technical users

## Quick Start

```bash
# Clone
git clone https://github.com/MoonGwanghoon/birkin.git
cd birkin

# Install (Python 3.11+)
pip install -e ".[dev]"

# Run
birkin-agent --help
```

## Requirements

- Python 3.11+
- Node.js 18+ (for browser tools)

## Development

```bash
# Install dev dependencies
pip install -e ".[dev]"

# Run tests
pytest

# Run agent
python run_agent.py --help
```
