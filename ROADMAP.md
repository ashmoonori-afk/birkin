# Birkin Roadmap

> This roadmap reflects our current priorities and direction. It is not a commitment — timelines and scope may shift as we learn from users and contributors.

---

## Phase 1: Foundation (Current)

**Goal:** Stable, self-hostable AI agent with messaging-first UX.

- [x] Core agent loop with multi-turn conversation
- [x] 40+ built-in tools (terminal, file, web search, browser, code execution)
- [x] Session persistence with full-text search
- [x] Telegram messaging gateway
- [x] Discord, Slack, WhatsApp adapters
- [x] CLI with interactive setup wizard
- [x] Skill system (bundled + community skills)
- [x] MCP integration
- [x] 200+ model providers via OpenRouter
- [x] Context compression for long conversations

## Phase 2: Polish & Community (Next)

**Goal:** Production-ready experience, contributor-friendly ecosystem.

- [ ] Landing page and documentation site
- [ ] Plugin marketplace / community skill registry
- [ ] Improved onboarding flow (< 5 min to first conversation)
- [ ] Agent memory improvements (long-term knowledge, user preferences)
- [ ] Subagent delegation and parallel task execution
- [ ] Scheduled tasks and automation workflows
- [ ] Docker one-command deployment
- [ ] Comprehensive test suite (80%+ coverage)

## Phase 3: Learning & Intelligence (Planned)

**Goal:** Agents that genuinely improve over time.

- [ ] Built-in skill creation from experience
- [ ] Reinforcement learning training pipeline
- [ ] Trajectory collection and evaluation
- [ ] Multi-agent collaboration patterns
- [ ] Knowledge graph for persistent agent memory

## Phase 4: Enterprise & Security (Planned)

**Goal:** Premium features for teams and businesses.

- [ ] Cybersecurity management harness (vulnerability scanning, threat detection)
- [ ] Team workspaces with role-based access control
- [ ] Analytics dashboard and audit logging
- [ ] Enterprise SSO integration
- [ ] Managed cloud hosting with SLA
- [ ] Security policy enforcement
