How It Works

Three steps to your personal AI

No technical background needed. Install, connect your apps, and start talking.

Step 1

Connect

Link Birkin to Telegram, Discord, Slack, or WhatsApp. Pick one or all of them — everything stays in sync.

Step 2

Ask

Talk to it like you would a colleague. Ask questions, give tasks, request research — in plain language.

Step 3

Grow

Birkin remembers what works and learns new skills over time. The more you use it, the better it gets.

Features

Everything you need, nothing you don't

Birkin handles the complexity so you can focus on what matters.

Lives Where You Do

Telegram, Discord, Slack, WhatsApp, and CLI from a single setup. Start a conversation on one, pick it up on another.

Gets Smarter Over Time

Persistent memory and auto-generated skills. It learns your preferences, your projects, and how you like things done.

Runs On a Schedule

Set up daily briefings, weekly reports, or automated backups with natural language. It runs while you sleep.

Handles Complex Tasks

Breaks big requests into smaller pieces, delegates to specialized sub-agents, and brings it all together.

Private and Secure

Self-hosted, open source, and under your control. Your data stays on your machine, never shared with third parties.

Web, Browser, and Beyond

Searches the web, reads pages, generates images, and automates browsers. It goes wherever the task takes it.

Get Started

Up and running in under a minute

One command to install, one to configure, one to start.

1

Install

curl -fsSL https://raw.githubusercontent.com/MoonGwanghoon/birkin/main/scripts/install.sh | bash

Works on Linux, macOS, and WSL2. No prerequisites needed.

2

Configure

birkin setup

Connect to OpenRouter, OpenAI, or your own endpoint.

3

Start talking

birkin

That's it. Full interactive AI with tools, memory, and learning built in.

Native Windows is experimental. Install WSL2 and run Birkin from there.

Open source, open book

Birkin is MIT-licensed and fully transparent. Read every line of code, host it yourself, contribute back. No black boxes, no lock-in, no surprises.
