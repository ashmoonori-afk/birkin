Help & Guide

Getting Started

Birkin is your personal AI assistant. Just type a message and press Enter to chat.

You can ask questions, get help with writing, brainstorm ideas, analyze information, and much more.

What Can Birkin Do?

  • Answer questions — Ask about any topic and get clear explanations
  • Write & edit — Draft emails, documents, messages, or creative content
  • Brainstorm — Generate ideas for projects, names, plans, and more
  • Summarize — Condense long text into key points
  • Analyze — Break down complex topics into simple language
  • Code help — Write, debug, or explain programming code

Tips for Best Results

  • Be specific about what you need — the more detail, the better the response
  • Ask follow-up questions to refine answers
  • Start a new conversation for a different topic to keep context clean
  • Use the sidebar to revisit past conversations

Managing Conversations

Click the menu icon (top-left) to open the sidebar. There you can:

  • See all your past conversations
  • Click any conversation to continue it
  • Start a fresh chat with the + button
  • Delete old conversations you no longer need

Keyboard Shortcuts

EnterSend your message
Shift+EnterAdd a new line
