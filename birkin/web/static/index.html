Help

Getting Started

Type a message and press Enter to chat with Birkin. Use @memory in your message to search your knowledge base.

Views

  • Chat — Conversation with AI (SSE streaming, tool calls, thinking indicator)
  • Workflow — Drag-and-drop workflow editor with 30+ node types
  • Memory — Knowledge graph (entities, concepts, sessions) — upload files or create pages
  • Telegram — Connect a Telegram bot (webhook or polling)
  • Triggers — Schedule workflows with cron, file watch, webhook, or message triggers
  • Skills — Browse and toggle installed skill plugins
  • Dashboard — Token spend, latency stats, error rates across providers
  • Approvals — Review and approve/reject pending agent actions
  • Insights — Weekly usage digest, provider distribution, cost tracking

Providers

9 LLM providers supported: Anthropic, OpenAI, Perplexity, Gemini, Ollama (local), Groq, OpenRouter, Claude CLI, Codex CLI. Switch in Settings.

Memory

Upload markdown or text files to Memory. The agent auto-saves important conversations. Use @memory search term in chat to search. Memory pages are injected into the system prompt for context.
